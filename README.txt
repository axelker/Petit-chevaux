
    Objectif du jeu : 
Le but du jeu est de faire se déplacer alternativement un cavalier de couleur rouge et un autre cavalier 
de couleur bleue sur un plateau. Chaque cavalier cherche à dépasser la ligne d’arrivée sur un parcours 
semé d’obstacles. A chaque tour un dé est roulé. Sa valeur indique le nombre de cases que le cavalier 
va franchir. 

Compilation : 

Lancer un terminal au sein du répetoir contenant src ,build et le script compilation.sh.
Executer le script via :  ./compilation.sh 
Si celui ci ne vous confère pas les droits : 
chmod +x compilation.sh 
ou
chmod +x ./compilation.sh


A l'execution la demande de la saisie de la longueur du plateau est demandé
Le plateau est ensuite afficher à l'initialisation puis la partie se déroule
jusqu'a ce que l'un des cavaliers se trouve sur la case d'arrivé.
